***📖 Description***

The Mali Digital Identity Assistant is an AI-powered web application that demonstrates how modern technologies such as Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), OCR, and vector databases can support digital identity management.

The application enables users to upload identity-related documents, extract information, search records, and interact with the data through a natural language AI assistant. While designed as a portfolio project, it reflects real-world concepts used in national digital identity platforms, citizen registration systems, and e-government initiatives.

***💡 What Is It?***

The Mali Digital Identity Assistant is a proof-of-concept platform built to showcase how Artificial Intelligence can improve digital identity services by combining document processing, intelligent search, and conversational AI.

Key capabilities include:

📄 Upload identity documents
🤖 AI-powered question answering
🔍 Intelligent document search
📑 OCR text extraction
🧠 Retrieval-Augmented Generation (RAG)
📊 Citizen record lookup
🌍 Modern web interface built with Streamlit

The project demonstrates how AI can help government agencies and organizations manage identity information more efficiently.

***⚙️ How Does It Work?****

The application follows a modern AI architecture:

Upload Documents
Users upload identity documents such as national IDs, passports, or registration forms.
Document Processing
The application extracts text using OCR and document parsing techniques.
Knowledge Indexing
Extracted information is converted into embeddings and stored in a vector database for semantic search.
AI Retrieval
When a user asks a question, the system retrieves the most relevant document sections.
LLM Response Generation
The retrieved context is provided to an OpenAI model, which generates accurate, grounded responses.
Interactive User Experience
Users receive natural language answers through an intuitive Streamlit interface.
